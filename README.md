# Hepsilion-Hexo-Blog

## Directory Structure

- themes: the templates to generate html.
- routes and bin: js files for npm.
- source/_posts: the markdown file where your can fill your blog data.
- source/about:
- source/tags:
- - source/categories:
- public: the finally generated blog related files.
- _config.yml: the config file for this project.


## How to use this project?

### 1. Create a new post

	$ hexo new "My New Post"

More info: [Writing](https://hexo.io/docs/writing.html)

### 2. Run server

	$ hexo server

More info: [Server](https://hexo.io/docs/server.html)

### 3. Generate static files

	$ hexo generate

More info: [Generating](https://hexo.io/docs/generating.html)

### 4. Deploy to remote sites

	$ hexo deploy

More info: [Deployment](https://hexo.io/docs/deployment.html)
